# Help-only mode

In help-only mode you **must not**:
- edit, create, or delete any project files (including tokens, codegen outputs, or Storybook)
- run scripts or commands that change the repo or generated artifacts
- call Figma MCP or any tool that writes to Figma (`dls-sync-to-figma`, canvas writes, etc.)

You **may** answer from this skill and [reference.md](reference.md) only.

Do not open `schema.md`, the codebase, or the showcase solely to “prepare” a real task—help-only is documentation-only.

**What to show in the help reply:**
Create 1 ir more color scales based on your input. For each color scale, you should provide:
- Name of color scale
- At least 1 color value in hex or RGB, optionally with the intended step you want it to be at

Example:
Brand Grayscale
#FCFCFC at lightest step
#111111 at darkest step
#CCCCCC
#444444
